# 个人网站练习项目

## 常用命令
- 启动本地开发环境
    ```shell
    npm start
    # or
    yarn start

- 测试命令
    ```shell
    npm test
    # or
    yarn test

- 打包命令
    ```shell
    npm run build
    # or
    yarn build

- 取出项目配置文件
    ```shell
    npm run eject
    # or
    yarn eject
